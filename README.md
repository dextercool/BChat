# BChat
Chat Realtime dengan Firebase. Silahkan dikembangkan gan, dan jangan lupa di Fork & Like :D

Dibuat menggunakan:
- Bootstrap
- jQuery
- Javascript
- Firebase
- PHP

Fitur:
- Realtime
- Send by Enter
- Safe Word
- Sound Notification
- Chat Count in Titlebar
- Send Link
- Send Image
